1. What is a computer 

A computer is a machine that can be programmed to automatically carry out sequences of arithmetic or logical operations.

2.  What is RAM?
RAM stands for Random Access Memory. It is a type of computer memory that is volatile, meaning its contents are lost when the computer is powered off. RAM is used by the computer's processor to store and quickly access data and instructions that are actively being used or processed.

3. Where is the storage in a computer?
Storage in a computer typically refers to non-volatile memory where data is stored persistently even when the computer is powered off. This includes hard disk drives (HDDs), solid-state drives (SSDs), and other forms of storage devices such as USB drives and memory cards.

4. What is the input device used to type text and numbers on a document in the computer system?
The input device used to type text and numbers on a document in a computer system is a keyboard.

5. What are the output devices?
Output devices are hardware components that display or present information generated by the computer. Common output devices include monitors (or screens), printers, speakers, and projectors.

6. Which is the input device that allows a user to move the cursor or pointer on the screen?
The input device that allows a user to move the cursor or pointer on the screen is a mouse or a touchpad.

7. Which language is directly understood by the computer without a translation program?
The language directly understood by the computer without a translation program is machine language or binary code. It consists of sequences of 0s and 1s representing instructions that the computer's central processing unit (CPU) can execute directly.

8. What are Input Devices?
Input devices are hardware components that allow users to interact with a computer system by providing data or instructions. Examples of input devices include keyboards, mice, touchpads, scanners, and microphones.
